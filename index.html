eJARN Collector Web Launcher

GitHub의 이 페이지는 로컬 PC에서 실행 중인 Streamlit 앱으로 이동하는 안내/실행 페이지입니다.

GitHub 저장소 열기
먼저 PC에서 Streamlit을 실행한 뒤, 로컬 Streamlit 열기 버튼을 눌러주세요.

사용 순서

  1. PC에서 프로젝트 폴더로 이동합니다.
  2. 가상환경을 활성화한 뒤 streamlit run streamlit_app.py를 실행합니다.
  3. 이 페이지에서 로컬 Streamlit 열기 버튼을 누릅니다.
  4. 열린 Streamlit 화면에서 수집을 실행하고 결과를 확인합니다.
  5. 결과 JSON은 프로젝트의 result/ 폴더에 저장됩니다.

실행 명령

cd jh104kim.github.io streamlit run streamlit_app.py
- GitHub 페이지는 Python 프로세스를 직접 실행하지 않습니다.
- 브라우저 보안 정책 때문에 GitHub 페이지에서 로컬 Streamlit을 iframe으로 안정적으로 임베드하는 방식은 권장되지 않습니다.
- 가장 안정적인 방식은 localhost:8501을 새 탭으로 여는 것입니다.
- 즉, GitHub의 index.html은 "실행 안내 + 로컬 앱 열기" 역할을 합니다.
- 현재 구조에서는 기존 Python 코드 수정 없이 이 방식으로 운영 가능합니다.
